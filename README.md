# twin-peaks-programming-language
twin peaks programming language

Требования:

Объявление переменных
```
    <identifier> type;
```

Присвание значения
```
    <identifier> = value;
```

Унарные Операции с перменными
```
    "!", "-"
```

Бинарные Операции с перменными
```
    "+", "-", "*", "/", "%"
```

## Базовые конструкции
Условные выржания
```
    if (expr) {
    
    } else {
    
    };
```

Цикл for 
```
   for (runOnce ; exitCondition ; runLoop) {

   }
```

## Функции
Определение функции (обязательно до использования)
```
    fn name(args type,...) <returnType> {return (optional)};
```

```
    name(args,...);
```

## Массивы
Объявление массива
```
    <identifier> type[N];
```

```
    <identifier>[x] = value;
    
    <other_identifier> = <identifier>[x];
```
